# Propuesta Innovación Digital — Presentación

Presentación HTML estática para la propuesta de Gestión de Flotas con IoT.

Cómo ejecutar localmente:

```bash
cd presentacion
python3 -m http.server 8000
# Abrir http://localhost:8000/presentacion.html
```

Archivos importantes:
- `presentacion.html` — archivo principal de la presentación.
- `assets/` — imágenes usadas por la presentación.

Despliegue rápido:
- Puedes activar GitHub Pages en la rama `main` (carpeta raíz) para servir `presentacion.html`.
